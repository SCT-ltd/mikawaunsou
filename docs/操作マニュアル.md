## 1. システムの概要

本システムは、三川運送の給与計算・勤怠管理を一元管理するためのウェブアプリケーションです。

### 主な機能

| 機能 | 内容 |
|------|------|
| 打刻管理 | 社員がスマートフォンから出退勤・休憩を記録 |
| 勤怠確認 | 事務所側でリアルタイムに全員の勤怠状況を把握 |
| 月次実績入力 | 出勤日数・走行距離・売上などを一覧入力 |
| 給与計算 | 月次実績をもとに給与を自動計算・明細作成 |
| メッセージ | 事務所と社員の間で連絡事項をやり取り |

### 利用するデバイス

- **事務所スタッフ（管理者）** → パソコンのブラウザで操作
- **ドライバー（社員）** → スマートフォンで専用打刻画面を使用

## 2. ログイン・ログアウト

### ログイン方法

1. ブラウザでシステムのURLを開く
2. 「ユーザー名」と「パスワード」を入力する
3. 「ログイン」ボタンを押す

> ⚠️ ユーザー名・パスワードが分からない場合は管理者に問い合わせてください。

### ログアウト方法

1. 画面右上の自分の名前をクリック
2. 「ログアウト」をクリック

> 💡 使い終わったら必ずログアウトしてください。

画面左側に縦に並んだメニューから各機能に移動できます。

```

💬 メッセージ
✅ 勤怠管理
📅 月次実績入力

🏷️ マスター管理
```

> - 「キャンセル（入力を続ける）」→ 元の画面に戻ります
> - 「保存せずに移動」→ 入力内容を破棄して移動します

## 4. ダッシュボード

ログイン後に最初に表示される画面です。会社全体の状況をひと目で確認できます。

### 表示される情報

- **今月の給与サマリー** — 全社員の総支給額・控除額の合計
- **出勤状況** — 現在の出勤中・休憩中・未出勤の人数
- **月次推移グラフ** — 過去数ヶ月の給与総額の推移
- **未処理の社員** — 給与明細がまだ作成されていない社員の一覧

## 5. ドライバー打刻アプリ（社員用・スマホ）

社員がスマートフォンから出退勤を記録するための専用画面です。

### アクセス方法

社員ごとに専用のURLが割り当てられています。QRコードを読み取るか、URLを直接開いてください。

> 例：`https://（システムURL）/driver/（社員番号）`

### PIN認証

画面を開くと、4桁のPINコード入力画面が表示されます。

1. テンキーで4桁のPINコードを入力する
2. 正しければ打刻画面に進む
3. 間違えると「PINコードが違います」と表示される

> 💡 PINコードは管理者から社員に個別に伝えてください。

---

### 打刻の手順

打刻画面は「勤怠」「チェックリスト」「メッセージ」の3つのタブに分かれています。

#### ① チェックリストの確認（出勤時のみ）

出勤前に「チェックリスト」タブを開き、車両の点検項目を確認します。

- 各項目を「良」または「否」にチェックする
- 「否」の箇所は出勤記録に記録されます（管理者が後で確認できます）

#### ② 出発地・到着地の入力（任意）

「勤怠」タブで出発地と到着地を入力できます（例：「本社 → 名古屋港」）。

#### ③ 走行距離（オドメーター）の入力

- **出勤時** → 「出発時オドメーター（km）」に現在のメーター値を入力
- **退勤時** → 「到着時オドメーター（km）」に現在のメーター値を入力

> 💡 オドメーターの入力値から走行距離が自動計算され、月次実績に反映されます。

#### ④ 打刻ボタンを押す

| ボタン | 押すタイミング |
|--------|--------------|
| **出勤** | 仕事を始めるとき |
| **退勤** | 仕事を終えるとき（オドメーター入力が必須） |
| **休憩開始** | 休憩に入るとき |
| **休憩終了** | 休憩から戻ったとき |

> ⚠️ 退勤ボタンは「到着時オドメーター」を入力しないと押せません。

> 📍 打刻と同時にGPS位置情報も自動記録されます。

---

### 打刻後の確認

打刻が成功すると、「出勤を記録しました」などのメッセージが画面上部に表示されます。画面中央には今日の打刻履歴が時系列で表示されます。

## 6. 勤怠管理

**メニュー：「勤怠管理」**

全社員の当日の出勤状況をリアルタイムで確認・管理する画面です。

### 画面の見方

- 上部のカード：出勤中・休憩中・退勤済・未出勤の人数
- テーブル：社員ごとの状態・出勤時刻・経過時間が一覧表示

### 日付の切り替え

画面上部のカレンダーボタンで過去の日付のデータも確認できます。

### 社員の詳細確認

テーブルの行をクリックすると、右側に選択した社員の詳細が表示されます。

- 出勤時刻・退勤時刻
- 打刻の種類ごとの一覧（出勤・休憩開始・休憩終了・退勤）
- 車両チェックリストの結果（否の項目がある場合は赤く表示）

### 打刻の修正・追加（管理者のみ）

社員の詳細パネルから、管理者が打刻を直接修正・追加・削除できます。

1. 社員名をクリックして詳細パネルを開く
2. 「打刻を追加」ボタンを押す、または既存の打刻の「編集」「削除」を行う
3. 種別と時刻を選んで保存する

> ⚠️ 修正は慎重に行ってください。打刻内容は月次実績の計算に影響します。

## 7. リアルタイムマップ

**メニュー：「リアルタイムマップ」**

出勤中のドライバーの現在位置を地図上でリアルタイムに確認できます。

- 緑のアイコン：現在位置
- 吹き出し：社員名・最終更新時刻

> 💡 ドライバーが打刻アプリを開いている間、15〜30秒ごとに位置が更新されます。

## 8. メッセージ

**メニュー：「メッセージ」**

事務所から社員へ、または社員から事務所へメッセージを送ることができます。

### 使い方

1. 左側のリストから社員を選ぶ
2. 画面下部のテキストボックスにメッセージを入力する
3. 「送信」ボタンを押す（またはEnterキー）

### 未読バッジ

メニューの「メッセージ」に赤い数字バッジが表示されている場合、未読メッセージがあります。

> 💡 社員側の打刻アプリにも「メッセージ」タブがあり、事務所からのメッセージを確認・返信できます。

## 9. 月次実績入力

**メニュー：「月次実績入力」**

給与計算の基礎となる各社員の月次データを入力・確認する画面です。

### 画面を開いたとき

月次実績入力画面を開くと、**勤怠管理の打刻データが自動的に反映されます**。

- 出勤日数（平日・土曜・日曜）
- 残業時間・欠勤日数
- 走行距離（オドメーターの差分合計）

手動で数値を変更することも可能です。

### 年月の切り替え

画面左上のプルダウンから年・月を選択します。

### 入力項目の説明

#### 勤怠・時間管理（青色列）

| 項目 | 説明 |
|------|------|
| 平日（日） | 月〜金の出勤日数。日給制社員の基本給計算に使用 |
| 土曜（日） | 土曜の出勤日数。土曜日給で計算 |
| 日曜（h） | 日曜・祝日の勤務時間（時間単位）。日曜時給で計算 |
| 欠勤（日） | 欠勤日数。欠勤控除の計算に使用 |
| 残業（h） | 法定外残業時間（時間単位）。×1.25で計算 |
| 深夜（h） | 深夜の勤務時間（時間単位）。+0.25割増 |
| 休日（日） | 法定休日の出勤日数。×1.35で計算 |

#### 運行実績（オレンジ色列）

| 項目 | 説明 |
|------|------|
| 走行KM | 当月の総走行距離。走行距離手当の計算に使用 |
| 件数 | 当月の配送件数。件数連動の手当計算に使用 |

#### 給与計算基礎（紫色列）

| 項目 | 説明 |
|------|------|
| 売上（円） | 当月の売上金額。歩合給の計算基礎 |
| 歩合% | 歩合率（%単位で入力。例：36.8） |
| BW売上（円） | Bluewing向けの売上金額 |

### 手動で再取り込みしたい場合

「**勤怠から一括反映**」ボタンを押すと、打刻データを再度読み込んで勤怠系の項目を上書きします。

### 保存方法

数値の確認が終わったら「**実績を保存**」ボタンを押してください。保存するまで変更内容はデータベースに記録されません。

> ⚠️ 保存前に別のページへ移動しようとすると、確認ダイアログが表示されます。

## 10. 給与明細

**メニュー：「給与明細」**

給与明細の作成・確認・印刷を行う画面です。

### 明細一覧

画面を開くと、選択した年月の全社員の給与明細が一覧で表示されます。

| 表示内容 | 説明 |
|----------|------|
| 社員名 | 氏名と所属部署 |
| 給与形態 | 月給制 / 日給制 |
| 総支給額 | 基本給＋各種手当の合計 |
| 控除合計 | 社会保険料＋所得税＋住民税の合計 |
| 差引支給額 | 手取り金額 |
| ステータス | 未計算 / 計算済 / 確定済 |

### 給与の計算

1. 月次実績入力が完了していることを確認する
2. 明細一覧の「計算」ボタンを押す（社員ごと、または一括）
3. 計算結果が画面に反映される

### 明細の詳細確認

社員名をクリックすると詳細画面が開き、以下を確認できます。

- 基本給の計算根拠（日給×出勤日数 など）
- 手当の内訳（残業手当・走行距離手当・深夜手当 など）
- 控除の内訳（健康保険・厚生年金・雇用保険・所得税・住民税 など）
- 差引支給額（手取り）

### 手当・控除の追加・編集

詳細画面の右側のパネルで、その社員だけの手当や控除を追加・削除できます。

1. 「手当を追加」または「差引を追加」ボタンを押す
2. 種別を選んで金額を入力する
3. 「手当・控除を保存」ボタンを押す

> 💡 定義されていない手当を追加したい場合は、先に「マスター管理」で手当の種類を作成してください。

### 給与明細の印刷・PDF出力

詳細画面の「印刷」ボタンを押すと、印刷用のレイアウトが表示されます。  
ブラウザの印刷機能（Ctrl+P）でPDF保存や印刷が可能です。

### 手入力モードで固定計算

「手入力固定で計算」ボタン（amber色）を使うと、自動計算ではなく手動で入力した金額をそのまま確定できます。

## 11. カレンダー

**メニュー：「カレンダー」**

社員ごとの年間出勤カレンダーを管理する画面です。

### 使い方

1. 社員を選択する
2. カレンダーの日付をクリックすると「出勤日↔休日」を切り替えられる
3. 赤色の日：祝日・土日
4. 黒色の日：平日（出勤日）

> 💡 このカレンダーは給与計算とは直接連動していません。勤務シフトの確認・共有用途で活用してください。

## 12. マスター管理（手当・控除）

**メニュー：「マスター管理」**

給与明細に表示される手当・控除の種類を定義する画面です。

### 手当の定義

「手当マスター」タブで、会社独自の手当を追加できます。

1. 「手当を追加」ボタンを押す
2. 名前・計算方法を設定する
3. 保存する

### 計算方法の種類

| 種類 | 説明 |
|------|------|
| 固定金額 | 毎月一定額を支給（例：皆勤手当 5,000円） |
| 走行距離連動 | 走行距離 × 単価（例：1kmあたり15円） |
| 件数連動 | 配送件数 × 単価（例：1件あたり100円） |
| 売上歩合 | 売上金額 × 歩合率 |

### 控除の定義

「控除マスター」タブで同様に控除項目を追加できます。

## 13. ユーザー管理

**メニュー：「ユーザー管理」**

システムにログインできるユーザーの管理を行います。

### ユーザーの追加

1. 「ユーザーを追加」ボタンを押す
2. ユーザー名・表示名・パスワード・権限を設定する
3. 保存する

### 権限の種類

| 権限 | できること |
|------|------------|
| **管理者（admin）** | 全機能の操作・設定変更・ユーザー管理 |
| **一般（staff）** | 給与計算・勤怠確認（設定変更は不可） |

### 社員のPINコード設定

社員が打刻アプリを使うためのPINコードもこの画面で設定します。

1. 社員の行をクリックして詳細を開く
2. 「PINコード」欄に4桁の数字を入力する
3. 保存する

## 14. 会社設定

**メニュー：「会社設定」**

給与計算に使う各種レートや会社情報を設定する画面です。

### 設定できる項目

#### 基本情報
- 会社名

#### 日給・時給単価
| 項目 | 説明 |
|------|------|
| 平日日給 | 日給制社員の平日1日あたりの賃金 |
| 土曜日給 | 日給制社員の土曜1日あたりの賃金 |
| 日曜時給 | 日曜・祝日の1時間あたりの賃金 |

#### 割増率
| 項目 | 標準値 | 説明 |
|------|--------|------|
| 残業割増率 | 1.25倍 | 法定外残業の割増（最低1.25） |
| 深夜割増率 | +0.25 | 深夜帯の追加割増 |
| 休日割増率 | 1.35倍 | 法定休日出勤の割増 |

#### 社会保険料率
| 項目 | 説明 |
|------|------|
| 健康保険（本人負担） | 給与に対する健康保険料率 |
| 厚生年金（本人負担） | 給与に対する厚生年金保険料率 |
| 雇用保険 | 給与に対する雇用保険料率 |

#### 計算設定
- 月間所定労働時間（時給換算の基準。標準160時間）

> ⚠️ 設定を変更した場合、変更後の月次分から新しい設定で計算されます。変更前の確定済み明細には影響しません。

## 15. よくある質問（FAQ）

### Q. 打刻が反映されない

**A.** 以下の点を確認してください。
- スマートフォンのインターネット接続が安定しているか
- 打刻後に「記録しました」のメッセージが表示されたか
- 位置情報の使用許可をブラウザに与えているか（拒否していると打刻できない場合があります）

---

### Q. 月次実績を保存したのに給与が変わらない

**A.** 月次実績を保存した後、給与明細画面で「計算」ボタンを押す必要があります。月次実績の保存だけでは給与は自動計算されません。

---

### Q. 手当を追加したいが選択肢がない

**A.** まず「マスター管理」画面で手当の種類を定義してください。定義した手当は給与明細の詳細画面から追加できるようになります。

---

### Q. 走行距離が自動で入力されない

**A.** 走行距離はドライバーが打刻時に「出発時オドメーター」と「到着時オドメーター」を入力した場合のみ自動計算されます。入力がない日は0kmとして集計されます。

---

### Q. 過去の打刻を修正したい

**A.** 「勤怠管理」画面で対象の社員を選び、詳細パネルの編集ボタンから修正できます。操作できるのは管理者権限を持つユーザーのみです。

---

### Q. 給与明細をPDFで保存したい

**A.** 給与明細の詳細画面で「印刷」ボタンを押し、印刷ダイアログで「PDFに保存」を選択してください（ブラウザの標準機能を使用します）。

---

### Q. パスワードを忘れた

**A.** 「ユーザー管理」画面から管理者がパスワードをリセットできます。管理者自身のパスワードを忘れた場合は、システム担当者にお問い合わせください。
