# 勤怠ページ フルモデルチェンジ設計書

日付: 2026-07-02
対象: `artifacts/payroll-app/src/pages/attendance.tsx`（現行 1,507 行の単一ファイル）
承認: ユーザー承認済み（マスターディテール統一・落ち着いた洗練系トーン）

## 目的

リアルタイム勤怠監視ダッシュボードを、monthly-input と統一した全幅マスターディテール型に刷新し、グラデーション/グロー多用の装飾を落ち着いた洗練系に整える。タイポグラフィも向上。**API・データ形状・全ロジック（SSE/ポーリング/ドラッグ並べ替え/GPS/集計/警告色）は変更しない**。

## レイアウト（`AppLayout fullWidth`）

- ツールバー: タイトル「勤怠」＋日付ナビ（前/カレンダー/次/今日）＋更新＋LIVEインジケータ
- 集計サマリー: 4タイル（出勤中/休憩中/未出勤/退勤済）。フラット（淡い色地＋大きな等幅数字＋状態ドット）。装飾円・グロー・グラデーションは撤去
- 2ペイン（md+）: 左＝社員リスト（固定幅）、右＝選択社員の詳細
- モバイル（<md）: リスト全画面 → 社員タップで詳細全画面（←戻る）

### 左ペイン: 社員リスト（employee-list.tsx）
- 検索（氏名・社員コード・部署）
- 各行: アバター / 氏名 / 社員コード / 状態バッジ / 出勤時刻 / 経過（8h橙・10h赤の警告色）/ 打刻数 / 欠勤・点検バッジ
- 選択ハイライト、Ctrl(⌘)+↑↓ で移動

### 右ペイン: 詳細（detail-panel.tsx）
現行スライドインパネルの内容を常設ペイン化:
- ヘッダー: アバター・氏名・社員コード・部署・状態・QRボタン
- 情報カード3枚: 出勤時刻 / 経過時間 / 休憩合計
- 補足カード: 最新発着地(📍) / 走行メーター(🚛) / 日常点検(✅⚠️)
- タイムライン: 欠勤・打刻エントリー（ドラッグ並べ替え＝楽観的更新→PATCH時刻交換、現行ロジックのまま）
- 打刻手動追加フォーム / 欠勤・休暇登録フォーム
- フッター: QR / 欠勤・休暇登録

## トーン（洗練系）
- グラデーション文字タイトル → 単色
- `shadow-[0_0_8px_...]` グロー・装飾円（`-top-4 -right-4 ...opacity-10`）撤去
- アバターは落ち着いた単色系
- 状態色（緑/琥珀/灰/空）は意味色として維持
- 角丸・境界線・余白・階層で整える

## タイポグラフィ
- 氏名・見出し: `jp-tight`（font-feature-settings palt）
- 時刻・経過・カウント: `amount` / tabular-nums
- Noto Sans JP は index.html で導入済み（monthly-input 対応時）
